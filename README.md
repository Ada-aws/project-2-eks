# Project 2 – Production-Ready Amazon EKS Deployment

## Overview
This project provisions a production-style Amazon EKS cluster using Terraform and deploys containerized microservices using Helm.

## Architecture Goals
- Infrastructure as Code (Terraform)
- Multi-AZ VPC networking
- Managed EKS Node Groups
- Helm-based deployments
- Horizontal Pod Autoscaling
- Git-based version control

## Tech Stack
- AWS
- Terraform
- Kubernetes (EKS)
- Helm
- Git
